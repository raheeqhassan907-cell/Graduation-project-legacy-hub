# مركز إرث لمشاريع التخرج - GP Legacy Hub 🎓 🚀

منصة تفاعلية متكاملة لتسجيل وأرشفة واستكشاف مشاريع التخرج الأكاديمية والبحثية، وتوفير دعم علمي ومهني مستدام للطلاب بتوجيه من الأساتذة والخريجين المتميزين.

---

## 🎨 الهوية البصرية والتصميم
* **ثيم زجاجي (Glassmorphism)**: يعتمد على بطاقات زجاجية شبه شفافة وحواف ناعمة بوهج أزرق أكاديمي متميز.
* **تجاوب كامل (Mobile Responsive)**: متوافق مع كافة مقاسات الشاشات والهواتف الذكية.
* **ثبات القائمة الجانبية (Sticky Sidebar)**: لتسهيل التنقل المستمر أثناء تصفح المحتوى.
* **تباين عالٍ ومقروئية ممتازة**: خطوط واضحة بلون كحلي داكن ورمادي فاتح تمنع أي تداخل بصري.

---

## 🛠️ أبرز الميزات البرمجية
1. **أرشفة المشاريع بالـ PDF**: إمكانية استعراض تفاصيل المشروع، التقنيات المستخدمة، المشرفين، وتحميل كتاب التخرج مباشرة.
2. **منتدى الأسئلة والأجوبة (Q&A Forum)**: مساحة للطلاب لطرح تساؤلاتهم الأكاديمية وتلقي الإرشادات من الأساتذة والخريجين.
3. **محرك التنبؤ والتحليل الذكي (AI Predictions & Trends)**: مدمج في الصفحة الرئيسية لتقديم توقعات ذكية مبنية على نماذج **Google Gemini 2.5 Flash API** حول المشاريع المستقبلية والتقنيات المطلوبة في سوق العمل.
4. **تكامل إرسال البريد الفعلي (Gmail SMTP)**: تفعيل استعادة كلمة المرور وإرسال كود تحقق حقيقي وآمن.
5. **مساعد إرث الذكي (Smart Chatbot)**: بوت دردشة عائم يحلل إحصائيات قاعدة البيانات ويبحث عن المشاريع ويجيب على الاستفسارات العامة.

---

## 💻 التقنيات المستخدمة
* **إطار العمل**: Laravel 11 (PHP 8.3+)
* **الواجهات والتنسيق**: HTML5 & Vanilla CSS (دون أطر عمل خارجية ثقيلة للسرعة الفائقة)
* **قاعدة البيانات**: SQLite / MySQL
* **محركات الذكاء الاصطناعي**: Google Gemini API
* **نظام التزامن**: Git & GitHub Web Hook (تزامن آلي لحظي للملفات)

---

## ⚙️ التثبيت والتشغيل المحلي

1. **تحميل المشروع وتثبيت الاعتماديات**:
   ```bash
   composer install
   npm install
   ```

2. **تهيئة ملف الإعدادات**:
   قم بإنشاء ملف `.env` وتهيئة قاعدة البيانات وإعدادات البريد الإلكتروني وربط مفتاح الـ API للذكاء الاصطناعي:
   ```env
   APP_NAME="مركز إرث لمشاريع التخرج"
   DB_CONNECTION=sqlite
   MAIL_MAILER=smtp
   MAIL_HOST=smtp.gmail.com
   MAIL_PORT=587
   MAIL_USERNAME=your_gmail@gmail.com
   MAIL_PASSWORD=your_app_password
   GEMINI_API_KEY=your_gemini_key
   ```

3. **تشغيل المهاجر والبيانات التجريبية**:
   ```bash
   php artisan migrate:refresh --seed
   ```

4. **تشغيل الخادم المحلي**:
   ```bash
   php artisan serve
   ```
   أو من خلال Laragon مباشرة عبر الرابط: `http://earth-grad-projects.test`
